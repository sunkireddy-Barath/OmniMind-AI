# Shared Types Package

This package contains shared TypeScript types and interfaces used across multiple services in the Decision Intelligence Platform. It aims to provide consistency and type safety throughout the application.

## Installation

To install the shared types package, run:

```bash
npm install @your-scope/shared-types
```

## Usage

Import the types and interfaces from the package in your TypeScript files as follows:

```typescript
import { YourType } from '@your-scope/shared-types';
```

## Contributing

If you would like to contribute to this package, please follow the standard contribution guidelines for the project. Ensure that any changes maintain the integrity of the types and interfaces provided.
