# UI Package Documentation

This package contains the user interface components for the Decision Intelligence Platform. It is built using Next.js 14, TypeScript, and TailwindCSS, providing a modern and responsive design.

## Installation

To install the necessary dependencies, run:

```
npm install
```

or

```
yarn install
```

## Usage

Import the components from this package in your Next.js application as needed. For example:

```typescript
import { MyComponent } from '@your-org/ui';
```

## Development

To start the development server, run:

```
npm run dev
```

or

```
yarn dev
```

This will start the Next.js application in development mode.

## Building

To build the package for production, run:

```
npm run build
```

or

```
yarn build
```
